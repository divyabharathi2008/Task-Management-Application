Task Management Application

A simple Task Management Application developed using HTML, CSS, and JavaScript.

Features

- Add new tasks
- Delete existing tasks
- Input validation for empty tasks
- Dynamic task management using JavaScript

Technologies Used

- HTML
- CSS
- JavaScript

How It Works

1. Enter a task in the input field.
2. Click the Add Task button.
3. The task is displayed in the task list.
4. Click the Delete button to remove a task.
5. An alert message is shown if the input field is empty.

Files

- "index.html" – User Interface
- "style.css" – Styling
- "script.js" – Task Management Logic
